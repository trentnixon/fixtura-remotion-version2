# Cricket Results — LLM design briefs (index)

Two static one-page briefs for design LLMs (no motion or implementation):

- **Association — both teams** — file `llm-brief-cricket-results-association.md` — Show **full results and player stats for home and away**.
- **Club — club stats only** — file `llm-brief-cricket-results-club.md` — Show **club player stats only**; opponent stays in the **team/score** band for context.

Shared rules: portrait artboard (e.g. 1080×1350), **minimum two fixtures per screen**, optional sponsor strip. Sample data: `testData/samples/Cricket/Cricket_Results.json`.
